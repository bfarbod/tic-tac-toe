# TIC TAC TOE
The tic tac toe game in python


## Getting Started


### Prerequisites

  * Python 2.7 

### How to run it
For Linux or Windows:<br />
    1. Go to the tic_tac_toe directory<br />
    2. Open your terminal or cmd<br />
    3. Run python<br />
      ```
      python tic_tac_toe.py
      ```
